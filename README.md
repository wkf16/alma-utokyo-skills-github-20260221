## 收录的 Skills

- `agile-study-protocol`：高强度迭代学习流程（发现问题、攻克薄弱点、变式训练、归档复盘）。
- `generate-creative-informatics-problem`：生成东大创造情报学风格的高难度练习题。
- `utokyo-p1-spartan-system`：P1（问题1）专项训练系统，强调领域追踪与训练日志。
- `utokyo-p3-tracker`：P3（问题3）贝叶斯策略跟踪器，评估题型掌握度与通过概率。
